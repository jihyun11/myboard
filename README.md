
https://www.h2database.com/html/download.html 사이트에 접속한다.
Platform-Independent Zip 파일을 다운받는다.  

아래의 명령어로 zip파일을 압축해제한다.
```agsl
tar -xvf h2-2023-09-17.zip
```
h2 실행 커맨드 권한설정 
```agsl
chmod 755 [h2설치경로]/bin/nh2.sh
```
h2 실행
```agsl
[h2설치경로]/bin/h2.sh
```
h2 접속
```agsl
localhost:8082
```
위의 주소로 접속후 h2에서 연결 버튼을 누르면 된다.
